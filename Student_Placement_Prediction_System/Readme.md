<b>Student Placement Prediction</b>
<p>
A Python-based data analysis project that explores a student placement dataset through data cleaning, preprocessing, exploratory data analysis (EDA), and visualization. The project aims to identify the factors that influence student placements and salary using statistical analysis and graphical representations.

<b>Features</b>
Data cleaning and preprocessing
Missing value handling
Duplicate removal
Exploratory Data Analysis (EDA)
Correlation analysis
Data visualization using Matplotlib and Seaborn
Insights into placement trends and salary relationships
Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Dataset Attributes
Gender
Department
City
CGPA
Aptitude Score
Communication Score
Attendance
Internship
Projects
Certifications
Placement Status
Salary (LPA)
Visualizations

The project includes several visualizations to better understand the dataset:

Scatter Plot (CGPA vs Aptitude Score)
Department-wise Student Distribution
City-wise Student Distribution
Placement Distribution (Pie Chart)
Correlation Heatmap
Box Plot
Histograms
Key Insights
Higher CGPA is associated with better placement opportunities.
Students with internships, certifications, and projects tend to have improved placement outcomes.
Attendance has a positive relationship with placement status.
Placement status shows a strong positive correlation with salary.
